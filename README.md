# Dotfiles Configuration

My personal configuration files for various development tools and terminal applications.

## Tools Included

### [Starship](https://starship.rs/)

Cross-shell prompt customization

- Configuration: `starship.toml`

### [Wezterm](https://wezfurlong.org/wezterm/)

GPU-accelerated cross-platform terminal emulator

### [Tmux](https://github.com/tmux/tmux/wiki)

Terminal multiplexer

### [Neovim](https://neovim.io/) with LazyVim distro

Hyperextensible Vim-based text editor
